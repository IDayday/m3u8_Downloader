## requirements

1. `python3.6+`
2. `requests` 
3. `ffmpeg`
    termux: `pkg install ffmpeg`  
    centos: `yum install ffmpeg`  
    mac os: `brew install ffmpeg`  
    ubuntu: `apt install ffmpeg`
   

## how to use

1. cmd running `python M3u8Download.py`
2. input `url` and `name` follow the tip

    `url`: m3u8 url
    `name`: video name
